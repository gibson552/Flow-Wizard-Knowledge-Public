Drop Columns
========

**Description:**

Remove specified columns from a table.

**Function:**

.. code-block:: python

    Drop_columns(table = string, cols = string)

**Parameters:**

- *Table:* Table name on which to perform function
- *Column(s):* Column(s) to drop

**Example:**

.. code-block:: python

   Drop_columns(table = Budget,  cols = ["Delta", "Plan")
