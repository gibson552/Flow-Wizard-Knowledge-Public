Append
========

**Description:**

Appends (Unions) on table to another.

**Function:**

.. code-block:: python

    Append(table = string, append_table = string, new_table_name = string)

**Parameters:**

- *Table:* Table name on which to perform function
- *Append Table:* Table to append to the parameter Table
- *New Table Name:* Name of the new table

**Example:**

.. code-block:: python

       Append(table = Budget, append_table = Budget_2, new_table_name = "Appended Table")
