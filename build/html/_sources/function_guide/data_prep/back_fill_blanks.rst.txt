Back Fill Blanks
========

**Description:**

Backward Propagation of Missing Data

**Function:**

.. code-block:: python

    Back_Fill_Blanks(table = string, cols = list)

**Parameters:**

- *Table:* Table name on which to perform function
- *Column(s):* Column(s) to backward fill

**Example:**

.. code-block:: python

   Back_Fill_Blanks(table = df, cols = ["Delta"])
